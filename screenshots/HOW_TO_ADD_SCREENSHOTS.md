## How to Take Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use `Print Screen` and paste in Paint

### macOS  
- Press `Cmd + Shift + 4` to select area
- Or `Cmd + Shift + 3` for full screen

### Linux
- Use `gnome-screenshot` or `scrot`

## Recommended Size
- **Width**: 1200-1400 pixels
- **Format**: PNG for best quality

## Tips
- Use a clean browser without extensions visible
- Show meaningful data in the screenshots
- Enable devmode before taking HEX View DEV screenshot
